Simple Selectors

Universal Selector

Everything.

*

Type Selector

Any element of type E.

E

Descendant Selector

Any element of type F that is a descendant of any element of type E.

E F

Common

Class Selector

Any element with class foo.

.foo
<div class="foo"></div>

ID Selector

Any element with ID foo.

#foo
<div id="foo"></div>

MOAR

Child Selector

Any element F that is direct-descendent child of E.

E > F
<E><F>foobar!</F></E>

Adjacent Selector

Any element F that is immediately preceded by a sibling E.

E + F
<E>foo</E> <F>bar</F>

Attributes!

Has an attribute foo.

E[foo]

Has an attribute foo that is set to bar.

E[foo="bar"]

Has an attribute foo that is space-separated tokens, one of which is bar.

E[foo~="bar"]

Has an attribute foo that is hyphen-separated tokens, and starts with bar.

E[foo|="bar"]

CSS2 Specification

  • count 1 if the declaration is from is a 'style' attribute rather than a rule with a selector, 0 otherwise
  • count the number of ID attributes in the selector
  • count the number of other attributes and pseudo-classes in the selector
  • count the number of element names and pseudo-elements in the selector
