Prototypal Inheritance

Classes vs. Prototypes

Classes
Type of data and functionality of objects.
Prototypes
Collections of behavior and structure for all instances.

When accessing the properties of an object, JavaScript will traverse the prototype chain upwards until it finds a property with the requested name.

  1. http://bonsaiden.github.com/JavaScript-Garden/#object.prototype
function Foo() {
    this.value = 42;
}
Foo.prototype = {
    method: function() {}
};

function Bar() {}

// Set Bar's prototype to a new instance of Foo
Bar.prototype = new Foo();
Bar.prototype.foo = 'Hello World';

// Make sure to list Bar as the actual constructor
Bar.prototype.constructor = Bar;

var test = new Bar() // create a new bar instance

// The resulting prototype chain
test [instance of Bar]
    Bar.prototype [instance of Foo]
        { foo: 'Hello World' }
        Foo.prototype
            { method: ... }
            Object.prototype
                { toString: ... /* etc. */ }
